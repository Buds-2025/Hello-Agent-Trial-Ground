# Hello Agent-Trial Ground

[简体中文](README.md) | [English](README_EN.md)

I adapted the 16 chapters of datawhalechina/hello-agents into an interactive Windows desktop game. Follow the chapter map to learn step by step, completing choices, sequencing exercises, and code diagnostics throughout the story. After finishing the game, you can also access the CG gallery, OST, and media center, including an opening cinematic, an ending theme, and 4K CG artwork.

## 1. Hardware Requirements

Any mainstream computer suitable for everyday office work should run the game smoothly.

## 2. How to Play

1. Go to the Releases page and download the latest game package.
2. Extract the entire package to any folder. A path without Chinese characters is recommended to avoid compatibility issues in some Windows environments.
3. Open the extracted folder and double-click `Hello Agent-Trial Ground.exe`. The game starts in borderless fullscreen mode by default. Press `F11` to switch to windowed mode. `Esc` returns to the previous screen.
4. The release package includes a completed save file for quickly previewing all content. The save location is `Documents\Hello Agent-Trial Ground\save\save.json`, which is created automatically after the desktop version runs successfully.
