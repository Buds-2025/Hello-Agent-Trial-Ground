# Hello Agent-Trial Ground

我把Hello-Agents的16章内容做成了一款互动游戏。你可以跟着地图逐章学习，在剧情中完成选择、排序和代码诊断；通关后还能打开CG与OST媒体中心。

## 下载

前往[Releases](../../releases)下载Windows版本，完整解压后运行`Hello Agent-Trial Ground.exe`。目前支持Windows 10和Windows 11的64位系统。

## 怎么玩

- 首次启动选择“开始新游戏”，在地图中按章节推进。
- 顶部可以跳过、存档或返回地图，右侧面板可以调整显示和声音。
- 存档位于`文档\Hello Agent-Trial Ground\save\save.json`。
- 完成全部16章后解锁CG入口。发行包里也附有全通关测试存档，方便快速预览。
