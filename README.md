# Hello Agent-Trial Ground

《Hello Agent-Trial Ground》是一款面向Agent学习与训练的互动视觉小说。游戏通过剧情、章节地图和多种互动练习组织16章内容，适合按照章节推进，也可以在通关后回看内容。

## 下载与运行

1. 在仓库的[Releases](../../releases)页面下载最新Windows版本。
2. 完整解压ZIP文件，不要只移动exe文件。
3. 双击`Hello Agent-Trial Ground.exe`启动游戏。

当前发行版本支持Windows 10和Windows 11的64位系统。

## 基本玩法

- 首次启动时选择“开始新游戏”。没有有效存档时，“继续游戏”保持禁用。
- 在章节地图中选择已经解锁的内容，阅读剧情并完成选择、排序、代码诊断等互动。
- 顶部功能区提供跳过、存档、地图等操作，右侧设置面板可以调整显示和声音。
- 桌面版进度保存在Windows“文档”目录下的`Hello Agent-Trial Ground\save\save.json`。
- 完成全部16章后，标题页的CG入口才会开放，可以查看写真、OST与媒体中心内容。

发行包附带一个全通关测试存档，用于快速预览章节及验证CG功能。压缩包中的README和存档说明提供了完整使用步骤。
